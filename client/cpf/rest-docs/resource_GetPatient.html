GetPatient

Given the id of a patient, return the patient details. The ID supplied may be the master ID or any of the aliases used for the patient. The ID returned in the results will be the master ID.

The following resources are applicable:

  • /patient/{patientNs}:{patientId}

/patient/{patientNs}:{patientId}
