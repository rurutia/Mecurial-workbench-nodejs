SearchDocuments

Search for documents and results from non-episodal sections or 'Diagnostics' section.

(TODO: review following restrictions) Current restrictions:

1. User can only choose to either return documents from non-episodal section or 'Diagnostics' section by specifying docSectionId='Diagnostics' parameters. There is no way documents/results from all sections can be returned together because we do not join tables at the moment

2. Less filters and sortable fields are supported when searching 'Diagnostics' compared to other non-episodal sections because current universal documentDTO is based on fields form Document object but not Result

The following resources are applicable:

  • /search/documents

Search for documents from NON-EPISODAL and Diagnostics sections.

patientId (no documentation provided) query

docSectionId document section id
  • When absent, return documents from non-episodal sections excluding 'Diagnostics'
  • When 'Diagnostics' is specified, only return documents and results from 'Diagnostics' section
